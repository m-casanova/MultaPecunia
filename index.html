Multa Pecunia

Una breve descrizione del font derivato da Titillium Pro Display

Multa Pecunia, un font variabile

Utilizzare le barre di scorrimento per testare il font (come indicato dall'Accademia della Crusca, si utilizza il maschile[1]).

Impostazioni:

wght (100...1000):

ital (0...1):

Descrizione

Titillium è un font realizzato dall'Accademia di Belle Arti di Urbino e distribuito con licenza OFL 1.1[2]; è ora distribuito via Google Fonts come Titillium Web in undici diverse versioni[3].

PagoPA ha creato Titillium Pro Display[4]. Il font variabile Multa Pecunia è una derivazione del font Titillium Pro Display, con metriche compatibili con Pass A38.

Il font può essere utilizzato tramite un unico file in formato WOFF2 con il seguente codice CSS.

@font-face {
    font-family: "Multa Pecunia";
    font-weight: 100 1000;
    src: url("MultaPecunia.woff2") format("woff2-variations");
}

Ovviamente il percorso del file deve essere adattato alla specifica situazione.

Per ottenere correttamente il corsivo con Chrome e Opera, utilizzare il seguente codice CSS.

em, i { font-variation-settings: "ital" 1; font-style:normal }
